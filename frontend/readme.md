## Instalación

* Cree un entorno virtual de python
```
python -m venv venv
```

* Active el entorno virtual
```
source venv/bin/activate
```

* Instale las dependencias 
```
pip install -r requirements.txt
```

## Como correr el proyecto

```
flask --app main.py run
```
